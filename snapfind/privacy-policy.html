개인정보 처리방침

최종 업데이트: 2025년 9월 2일
요약: 스냅파인드는 사용자가 촬영한 사진에 라벨을 붙이고 빠르게 검색할 수 있도록 돕는 오프라인 중심 앱입니다. 개인정보를 별도로 수집하거나 외부로 전송하지 않으며, 사진, 라벨, 검색 색인은 사용자 기기 내에서만 처리·저장됩니다.

1. 개인정보 수집 및 이용

스냅파인드(이하 "앱")는 다음과 같이 개인정보를 처리합니다:

  • 수집하는 개인정보: 없음
  • 회원가입, 로그인 등 개인 식별을 위한 기능이 없습니다
  • 사용자 정보를 외부 서버로 전송하거나 제3자와 공유하지 않습니다

2. 처리되는 데이터의 범위

앱 기능을 위해 다음과 같은 사용자 생성 데이터가 기기 내에서만 처리됩니다:

  • 사진/이미지: 사용자가 촬영하거나 선택한 이미지
  • 라벨 및 메모: 사용자가 사진에 부여한 태그, 제목, 설명 등
  • 검색 색인: 빠른 검색을 위한 기기 내 색인 데이터

이 데이터는 앱 기능 제공을 위해서만 사용되며 외부 전송이 이루어지지 않습니다.

3. 데이터 저장 및 보관

  • 모든 사진, 라벨, 검색 색인은 사용자의 기기 내부에만 저장됩니다
  • 클라우드 동기화나 서버 업로드 기능을 제공하지 않습니다
  • 앱 삭제 또는 사용자가 데이터를 삭제할 경우 관련 데이터는 기기에서 제거됩니다
  • 운영체제 또는 사용자가 선택한 백업 솔루션(iCloud/Google 백업 등)에 의해 기기 백업이 이루어질 수 있으나, 이는 OS/백업 서비스 정책에 따릅니다

4. 인터넷 연결 및 데이터 전송

  • 앱은 기본적으로 오프라인에서 동작하며, 앱 기능 수행을 위한 사용자 개인정보 전송이 없습니다
  • 앱 업데이트 확인 등 스토어 관련 통신을 제외하고, 사용자 데이터가 네트워크로 전송되지 않습니다

5. 제3자 정보 공유

  • 광고 SDK, 분석 도구, 소셜 로그인 등 제3자 서비스를 사용하지 않습니다
  • 사용자 사진과 라벨 정보를 제3자에게 제공하지 않습니다

6. 앱 권한

앱은 다음 권한을 요청할 수 있습니다:

  • 카메라: 사진 촬영을 위해 필요
  • 사진/미디어/파일(갤러리 접근): 기기 내 사진을 선택·저장하기 위해 필요

앱은 기능 제공에 불필요한 권한을 요청하지 않습니다.

7. 데이터 보안

  • 데이터는 기기 내부에서만 처리되며 외부 전송이 없습니다
  • 운영체제에서 제공하는 보안 메커니즘(암호화 저장소 등)을 최대한 활용합니다

8. 사용자 권리

  • 앱 내에서 사진, 라벨 등 데이터를 직접 수정·삭제할 수 있습니다
  • 앱을 삭제하면 앱이 보관하던 데이터는 기기에서 제거됩니다

9. 아동의 개인정보

본 앱은 모든 연령 사용자가 사용할 수 있으나, 별도의 개인정보 수집 기능(회원가입/로그인 등)이 없습니다. 아동의 개인정보를 고의로 수집하지 않습니다.

10. 개인정보 처리방침 변경

  • 본 방침이 변경될 경우, 이 페이지를 통해 고지합니다
  • 중대한 변경 시 앱 업데이트 시점에 안내합니다
  • 변경된 방침은 게시일로부터 효력이 발생합니다
