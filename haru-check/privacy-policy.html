개인정보 처리방침

최종 업데이트: 2025년 1월 27일
요약: 하루체크 앱은 사용자의 개인정보를 수집하지 않습니다. 몸무게 체크 기능을 위해 사용자가 직접 입력하는 데이터는 앱 기능 데이터로, 모든 것이 사용자 기기 내부에만 저장됩니다. 외부 서버로 전송되지 않으며, 다만 광고 표시를 위해 제3자 광고 서비스가 사용될 수 있습니다.

1. 개인정보 수집 및 이용

본 하루체크 앱("앱")은 다음과 같이 개인정보를 처리합니다:

  • 수집하는 개인정보: 없음
  • 앱은 사용자의 개인정보를 수집하지 않습니다
  • 로그인 기능이 없으며, 사용자 식별 정보를 요구하지 않습니다
  • 사용자의 개인정보를 외부 서버로 전송하지 않습니다

1.1 앱 기능 데이터

몸무게 체크 기능을 위해 사용자가 직접 입력하는 데이터:

  • 몸무게 측정값, 측정 날짜, 메모 등의 체중 관리 정보
  • 차트 및 진행률 표시를 위한 통계 데이터
  • 이러한 데이터는 개인정보가 아닌 앱 기능 데이터로, 사용자 기기 내부에만 저장됩니다
  • 외부 서버로 전송되지 않으며, 개발자나 제3자가 접근할 수 없습니다

2. 앱 기능 데이터 저장 및 보관

몸무게 체크 기능 데이터 처리에 대해:

  • 모든 몸무게 체크 데이터는 사용자의 기기 내부에만 저장됩니다
  • 외부 서버나 클라우드에 데이터를 업로드하지 않습니다
  • 데이터는 기기의 로컬 데이터베이스에 안전하게 저장됩니다
  • 앱 삭제 시 모든 데이터가 함께 삭제됩니다
  • 이러한 데이터는 개인정보가 아닌 앱 기능 데이터입니다

3. 인터넷 연결 및 데이터 전송

앱 기능을 위해 필요한 데이터 전송:

  • 앱 업데이트: 새로운 기능이나 보안 패치를 위해 앱스토어 연결이 필요할 수 있습니다
  • 몸무게 체크 데이터는 외부로 전송되지 않습니다
  • 사용자 개인정보는 포함되지 않습니다

4. 제3자 정보 공유

본 앱은 사용자의 개인정보를 제3자와 공유하지 않습니다. 다만, 광고 표시를 위해 다음과 같은 서비스가 사용될 수 있습니다:

  • 광고 네트워크: 앱 내 광고 표시를 위해 제3자 광고 서비스가 사용됩니다
  • 분석 도구 사용 안 함
  • 소셜미디어 연동 안 함
  • 사용자 개인정보를 외부로 전송하지 않음

4.1 광고 서비스

앱에서 표시되는 광고와 관련하여:

  • 광고 서비스 제공업체는 자체 개인정보처리방침을 가지고 있습니다
  • 광고 표시를 위해 기기 식별자(광고 ID)가 사용될 수 있습니다
  • 광고 관련 데이터는 광고 서비스 제공업체의 정책에 따라 처리됩니다
  • 사용자는 기기 설정에서 광고 ID를 재설정하거나 제한할 수 있습니다

5. 앱 권한

본 앱이 요청하는 권한:

  • 저장소 접근: 몸무게 체크 데이터를 기기에 저장하기 위함
  • 기타 불필요한 권한은 요청하지 않습니다

6. 데이터 보안

앱 기능 데이터의 보안을 위해:

  • 모든 데이터는 기기 내부에서만 처리됩니다
  • 데이터는 암호화되어 저장됩니다
  • 외부 네트워크를 통한 데이터 전송이 없습니다
  • 정기적인 보안 업데이트를 제공합니다
  • 개인정보를 수집하지 않으므로 개인정보 유출 위험이 없습니다
⚠️ 주의사항: 본 앱은 건강 관련 정보를 다루지만, 의료적 진단이나 치료 목적으로는 사용하지 마세요. 건강에 대한 우려사항이 있으시면 의료 전문가와 상담하시기 바랍니다.

본 앱은 모든 연령의 사용자가 사용할 수 있습니다:

  • 아동의 개인정보를 수집하지 않습니다
