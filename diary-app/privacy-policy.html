최종 업데이트: 2025년 5월 27일
요약: 본 일기장 앱은 사용자의 개인정보를 수집하거나 외부로 전송하지 않습니다. 모든 데이터는 사용자의 기기에만 저장됩니다.

1. 개인정보 수집 및 이용

본 일기장 앱("앱")은 다음과 같이 개인정보를 처리합니다:

  • 수집하는 개인정보: 없음

  • 로그인 기능이 없으며, 사용자 식별 정보를 요구하지 않습니다
  • 네트워크 연결을 통한 데이터 전송을 하지 않습니다

2. 일기 데이터 저장 및 보관

사용자가 작성하는 일기 내용에 대해:

  • 모든 일기 데이터는 사용자의 기기 내부에만 저장됩니다
  • 외부 서버로 전송되지 않습니다
  • 개발자나 제3자가 접근할 수 없습니다
  • 앱 삭제 시 모든 데이터가 함께 삭제됩니다

3. 제3자 정보 공유

본 앱은 어떠한 정보도 제3자와 공유하지 않습니다:

  • 광고 네트워크 사용 안 함
  • 분석 도구 사용 안 함
  • 소셜미디어 연동 안 함
  • 클라우드 서비스 연동 안 함

4. 앱 권한

본 앱이 요청하는 권한:

  • 저장소 접근: 일기 데이터를 기기에 저장하기 위함
  • 기타 불필요한 권한은 요청하지 않습니다

5. 데이터 보안

사용자 데이터의 보안을 위해:

  • 모든 데이터는 기기 내부에서만 처리됩니다
  • 네트워크를 통한 데이터 전송이 없어 해킹 위험이 없습니다
  • 사용자가 직접 데이터를 관리할 수 있습니다

본 앱은 만 13세 미만 아동의 개인정보를 수집하지 않습니다. 실제로 어떠한 개인정보도 수집하지 않으므로 모든 연령대에서 안전하게 사용할 수 있습니다.
