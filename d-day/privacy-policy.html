개인정보 처리방침

최종 업데이트: 2025년 7월 27일
요약: 디데이 앱은 사용자의 개인정보를 수집하지 않습니다. 디데이 관리, 다이어리 작성, 투두리스트 기능과 안드로이드 위젯 설정을 위해 필요한 데이터만 기기 내부에 저장되며, 외부로 전송되지 않습니다.

1. 개인정보 수집 및 이용

디데이 앱(이하 "앱")은 사용자의 개인정보를 수집하지 않습니다:

  • 수집하는 개인정보: 없음
  • 앱은 사용자가 직접 입력하는 디데이 정보, 다이어리 내용, 투두리스트만 저장합니다
  • 로그인, 회원가입 등 개인 인증 기능이 없습니다
  • 사용자의 개인정보를 외부 서버로 전송하지 않습니다

2. 데이터 저장 및 보관

앱 내 데이터 처리에 대해:

  • 모든 디데이 정보, 다이어리 내용, 투두리스트는 사용자의 기기 내부에만 저장됩니다
  • 외부 서버나 클라우드에 데이터를 업로드하지 않습니다
  • 안드로이드 위젯 설정 정보도 기기 내부에만 저장됩니다
  • 앱 삭제 시 모든 데이터가 함께 삭제됩니다

3. 인터넷 연결 및 데이터 전송

앱 기능을 위해 필요한 데이터 전송:

  • 앱 업데이트: 새로운 기능이나 보안 패치를 위해 앱스토어 연결이 필요할 수 있습니다
  • 디데이 관련 데이터는 외부로 전송되지 않습니다
  • 사용자 개인정보는 포함되지 않습니다

4. 제3자 정보 공유

본 앱은 어떠한 사용자 정보도 제3자와 공유하지 않습니다:

  • 광고 네트워크 사용 안 함
  • 분석 도구 사용 안 함
  • 소셜미디어 연동 안 함
  • 사용자 개인정보를 외부로 전송하지 않음

5. 앱 권한

본 앱이 요청하는 권한:

  • 저장소 접근: 디데이 정보, 다이어리, 투두리스트 저장 위함
  • 기타 불필요한 권한은 요청하지 않습니다

6. 데이터 보안

사용자 데이터의 보안을 위해:

  • 모든 데이터는 기기 내부에서만 처리됩니다
  • 외부 네트워크를 통한 데이터 전송이 없습니다
  • 사용자가 직접 데이터를 관리할 수 있습니다

7. 사용자 연령 제한

본 앱은 모든 연령의 사용자가 사용할 수 있습니다. 아동의 개인정보를 수집하지 않습니다.
