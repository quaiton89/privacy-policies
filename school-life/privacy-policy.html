개인정보 처리방침

최종 업데이트: 2025년 6월 24일
요약: 본 스쿨라이프 앱은 사용자의 개인정보를 수집하거나 외부로 전송하지 않습니다. 학교 급식, 시간표, 일정 정보 조회를 위해 인터넷 연결이 필요하지만, 사용자가 입력한 개인정보는 기기 내부에만 저장됩니다. 본 앱은 만 13세 이상 사용자를 대상으로 합니다.

1. 개인정보 수집 및 이용

본 스쿨라이프 앱("앱")은 다음과 같이 개인정보를 처리합니다:

  • 수집하는 개인정보: 없음
  • 앱은 사용자의 개인정보를 어떠한 형태로도 수집하지 않습니다
  • 로그인 기능이 없으며, 사용자 식별 정보를 요구하지 않습니다
  • 사용자가 입력한 정보는 기기 내부에만 저장됩니다

2. 입력받는 정보 및 데이터 처리

앱 기능을 위해 필요한 정보 입력 및 처리:

  • 사용자 입력 정보: 이름, 학교명, 반 정보 등 앱 설정을 위한 기본 정보
  • 학교 정보 조회: 공개된 학교 API를 통한 급식, 시간표, 일정 정보 조회
  • 입력받은 정보는 모두 사용자의 기기 내부에만 저장됩니다
  • 외부 서버로 전송되지 않으며, 개발자나 제3자가 접근할 수 없습니다

3. 앱 데이터 저장 및 보관

앱 내 데이터 처리에 대해:

  • 사용자가 입력한 설정 정보는 사용자의 기기 내부에만 저장됩니다
  • 조회한 학교 정보는 임시로 기기 내부에 저장됩니다
  • 사용자 개인정보는 외부로 전송되지 않습니다
  • 앱 삭제 시 모든 저장된 데이터가 함께 삭제됩니다

4. 제3자 정보 공유

본 앱은 어떠한 사용자 정보도 제3자와 공유하지 않습니다:

  • 광고 네트워크 사용 안 함
  • 분석 도구 사용 안 함
  • 소셜미디어 연동 안 함
  • 사용자 개인정보를 외부로 전송하지 않음

5. 앱 권한

본 앱이 요청하는 권한:

  • 인터넷 연결: 학교 정보 API 조회를 위해 필요
  • 저장소 접근: 앱 설정 및 임시 데이터 저장을 위해 필요

6. 데이터 보안

사용자 데이터 보호를 위한 조치:

  • 모든 데이터는 사용자 기기 내부에만 저장
  • 외부 서버와의 통신 시에도 개인정보 전송 금지
  • 데이터 암호화 및 보안 강화

7. 사용자 권리

사용자는 다음과 같은 권리를 가집니다:

  • 앱 내 저장된 모든 데이터 삭제 권한
  • 앱 설정 초기화 권한
  • 앱 삭제를 통한 모든 데이터 완전 제거

8. 개인정보처리방침 변경

본 개인정보처리방침은 법령 및 방침에 따라 변경될 수 있으며, 변경 시 앱 내 공지사항을 통해 고지합니다.
