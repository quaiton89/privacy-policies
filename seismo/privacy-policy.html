개인정보처리방침

시즈모(Seismo) 앱은 전세계 지진 정보를 제공하는 서비스로, 사용자의 개인정보를 최소한으로만 수집하며 로컬 기기 내에서만 데이터를 처리합니다. 위치 정보는 지진 정보 표시 목적으로만 사용되며 외부로 전송되지 않습니다.

1. 개인정보 수집 및 이용

시즈모 앱은 다음과 같은 개인정보를 수집하지 않습니다:

  • 개인 식별 정보 (이름, 전화번호, 이메일 등)
  • 기기 정보
  • 사용 패턴 데이터

1.1 위치 정보

앱에서 제공하는 '현재 위치' 기능은 다음과 같이 처리됩니다:

  • 사용자의 현재 위치는 지진 정보를 표시하기 위한 목적으로만 사용됩니다
  • 위치 정보는 사용자 기기 내부에만 저장되며, 외부 서버로 전송되지 않습니다
  • 위치 정보는 앱을 종료하거나 기기를 재시작하면 자동으로 삭제됩니다
  • 사용자는 언제든지 기기 설정에서 위치 권한을 해제할 수 있습니다

2. 데이터 처리 방식

2.1 지진 데이터

앱에서 제공하는 지진 정보는 공개된 지진 데이터베이스에서 가져온 것으로, 다음과 같이 처리됩니다:

  • 모든 지진 데이터는 사용자의 기기 내부에만 저장됩니다
  • 외부 서버로 개인정보나 사용 데이터를 전송하지 않습니다
  • 데이터는 오프라인에서도 확인할 수 있도록 로컬에 캐시됩니다

2.2 앱 설정 데이터

사용자가 설정한 앱 내 옵션들(언어 설정, 알림 설정 등)은 기기 내부에만 저장되며, 외부로 전송되지 않습니다.

3. 제3자 서비스

시즈모 앱은 다음과 같은 제3자 서비스를 사용할 수 있습니다:

  • 지진 데이터 제공 서비스: 공개 API를 통해 지진 정보를 수집합니다
  • 광고 서비스: 무료 버전의 경우 광고가 표시될 수 있습니다

이러한 서비스들은 자체적인 개인정보처리방침을 가지고 있으며, 해당 서비스의 정책이 적용됩니다.

4. 데이터 보안

개인정보를 수집하지 않으므로 별도의 데이터 보안 조치가 필요하지 않습니다. 다만, 앱의 안정성과 사용자 경험 향상을 위해 다음과 같은 기술적 조치를 취합니다:

  • 로컬 데이터의 안전한 저장
  • 앱 업데이트를 통한 보안 강화
  • 사용자 기기의 기본 보안 설정 준수

5. 사용자의 권리

시즈모 앱은 개인정보를 최소한으로만 수집하므로 다음과 같은 권리가 적용됩니다:

  • 언제든지 앱을 삭제하여 모든 데이터를 제거할 수 있습니다
  • 개인정보 처리에 대한 문의를 할 수 있습니다
  • 본 처리방침에 동의하지 않을 경우 앱 사용을 중단할 수 있습니다

6. 개인정보처리방침 변경
