개인정보 처리방침

최종 업데이트: 2025년 8월 11일
요약: 뉴스픽 앱은 사용자의 개인정보를 수집하거나 저장하지 않습니다. 뉴스 콘텐츠는 RSS 및 공개 API 호출을 통해서만 가져오며, 앱은 광고·분석 도구·소셜 연동을 사용하지 않습니다.

1. 개인정보 수집 및 이용

뉴스픽 앱(이하 "앱")은 다음과 같이 개인정보를 처리합니다:

  • 수집하는 개인정보: 없음
  • 로그인, 회원가입 등 개인 인증 기능이 없습니다
  • 사용자의 개인정보를 외부로 전송하거나 서버에 저장하지 않습니다

2. 데이터 저장 및 보관

앱은 개인을 식별할 수 있는 데이터를 저장하지 않습니다. 앱 설정(예: 보기 옵션, 즐겨찾기 등)이 있는 경우, 해당 정보는 사용자의 기기 내부에만 저장되며 외부 서버로 전송되지 않습니다.

3. 인터넷 연결 및 데이터 전송

앱은 뉴스 콘텐츠 제공을 위해 네트워크 요청을 수행합니다:

  • RSS/공개 API 호출: 뉴스 기사 제목, 요약, 썸네일 등 공개 콘텐츠만 요청합니다
  • 요청 수행 시 제3자 서버로 IP 주소, User-Agent, 요청 시각 등 표준 기술 정보가 전달될 수 있으나, 이는 콘텐츠 제공을 위한 통신 과정에서 발생하는 일반적인 범위입니다
  • 앱은 위 기술 정보를 수집·저장·추적하지 않습니다
  • 각 RSS 제공처 또는 API 제공자의 정책은 해당 제공처의 개인정보 처리방침을 따릅니다

4. 제3자 정보 공유

본 앱은 사용자 정보를 제3자와 공유하지 않습니다:

  • 광고 네트워크 사용 안 함
  • 분석 도구 사용 안 함
  • 소셜 로그인/SDK 사용 안 함
  • 사용자 개인정보를 외부로 전송하지 않음

5. 앱 권한

본 앱이 요청하는 권한:

  • 네트워크 접근: RSS 및 공개 API에서 콘텐츠를 가져오기 위함
  • 파일, 위치, 연락처, 카메라 등 불필요한 권한은 요청하지 않습니다

6. 데이터 보안

앱은 개인정보를 저장하지 않으며, 앱 내 설정 데이터는 기기 내부 저장소에서만 처리됩니다. 외부 네트워크를 통한 사용자 개인정보 전송은 없습니다.

7. 사용자 연령

본 앱은 모든 연령의 사용자가 사용할 수 있습니다. 아동의 개인정보를 수집하지 않습니다.

8. 개인정보 처리방침 변경

본 처리방침이 변경될 경우:

  • 변경 사항은 이 페이지에 게시됩니다
  • 중요한 변경사항이 있을 경우 앱 업데이트를 통해 알려드립니다
  • 변경된 처리방침은 게시일로부터 효력이 발생합니다

9. 사용자 권리

사용자는 다음 권리를 가집니다:

  • 언제든지 앱을 삭제하여 모든 로컬 데이터를 제거할 수 있습니다
  • 개인정보 처리에 대한 문의를 할 수 있습니다
  • 본 처리방침에 동의하지 않을 경우 앱 사용을 중단할 수 있습니다
